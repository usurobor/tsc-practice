# Coherent Agent Process (CAP)

CAP is the **self-governance spec**: how a single agent runs its own behavior loop coherently under TSC/CT/HOS.

If CLP is about *artifacts* (how you iterate a spec or post), CAP is about **process**:

- how you interpret instructions as Terms,
- how you plan and execute work,
- how you report back to your human,
- when you Brake and how you Exit.
